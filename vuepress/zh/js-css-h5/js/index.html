# 为什么叫 javaScript？最初叫 LiveScript 。
  • 1995 Netscape公司开发，借鉴了java语言(当时很火)，希望看起来像java，后来才改名的。
  • JavaScript的标准是ECMAScript。ECMAScript2015也叫es6（第六版） 。
  • ECMAS：欧洲计算机制造协会，联合各家（微软等）统一标准。
