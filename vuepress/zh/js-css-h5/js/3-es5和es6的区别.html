es5和es6的区别

# 主要区别 详见 es6 的扩展

    1. let命令 ES6中的let命令，声明变量，用法和var差不多，但是let是为JavaScript新增了块级作用域，ES5中是没有块级作用域的，并且var有变量提升的概念，但是在let中，使用的变量一定要进行声明

    1. 解构赋值 ES6中变量的解构赋值，比如：var [a,b,c] = [0,1,2];
    1. 箭头函数 ES6中的函数定义也不再使用关键字function，而是利用了()=>来进行定义
    1. 函数默认参数 ES6中可以设置默认函数参数，如function A（x,y=9）{};
    1. class构造函数 实现面向对象编程的新形式
  • class是一种新的语法形式，是class name{…}这种形式和函数的写法完全不同；
  • class的{}区间内，只能写构造器，静态方法，静态属性，实例方法；
  • class创建的构造函数，函数体要放在class中的constructor函数中，constructor即构造器，初始化实例的默认行为；
  • class中函数的写法是add(){…}这种形式，并没有function关键字。
//  例子

class myClass{
    constructor(x,y){
        this.x=x;
        this.y=y;
    }
    add(){
        return this.x + this.y
    }
}

const mynewClass=new myClass(1,2)
console.log(mynewClass.add()) 
// 输出 3
