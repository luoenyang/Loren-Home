# async、await、promise

# async和await的底层实现

  1. async函数的转换：
    当JavaScript引擎遇到一个async函数时，它会自动将这个函数转换为一个返回Promise对象的函数。这个返回的Promise对象会管理async函数内部的异步操作。可以在async函数内部使用await表达式来暂停异步操作的执行，直到Promise解析完成，然后继续执行async函数并返回结果。

  2. await表达式：
    await关键字用于等待一个Promise对象。它只能在async函数内部使用。当await一个Promise时，它会暂停async函数的执行，直到Promise解决（resolve）或拒绝（reject）。如果Promise被解决，await表达式的结果就是Promise解决时的值。

# promise底层实现

# Promise的基本结构

  1. 构造函数 Promise是一个构造函数，用于创建一个新的Promise实例。它接收一个执行器函数（executor function）作为参数，这个执行器函数有两个参数：resolve和reject，它们都是函数，分别用于将Promise的状态从pending（进行中）改为fulfilled（已成功）或rejected（已失败）。

  2. 状态管理：pending（进行中）、fulfilled（已成功）、rejected（已失败）；只能从pending变为fulfilled或rejected，且一旦状态改变，就不能再变。

  3. 结果数据：通过resolve和reject返回。

# Promise的底层实现

  1. 实例属性：status（状态）和value/reason
  2. 内部函数：_resolve和_reject，用于更新Promise的状态和结果数据。
  3. 执行器函数的执行：Promise构造函数被调用时，会立即执行传入的执行器函数，并传入_resolve和_reject作为参数。
  4. then方法的实现：promise原型上定义了then方法，用于成功和失败的回调函数。then方法返回一个新的Promise实例，以支持链式调用。
  5. 微任务队列：promise的回调函数（包括then、catch等）会被添加到微任务队列中，等待当前执行栈清空后执行。
class MyPromise {  
    constructor(executor) {  
        this.state = 'pending'; // 初始状态为pending  
        this.value = undefined; // 成功的结果  
        this.reason = undefined; // 失败的原因  
        this.onFulfilledCallbacks = []; // 成功回调的数组  
        this.onRejectedCallbacks = []; // 失败回调的数组  
  
        const resolve = (value) => {  
            if (this.state === 'pending') {  
                this.state = 'fulfilled';  
                this.value = value;  
                this.onFulfilledCallbacks.forEach(fn => fn());  
            }  
        };  
  
        const reject = (reason) => {  
            if (this.state === 'pending') {  
                this.state = 'rejected';  
                this.reason = reason;  
                this.onRejectedCallbacks.forEach(fn => fn());  
            }  
        };  
  
        try {  
            executor(resolve, reject);  
        } catch (error) {  
            reject(error);  
        }  
    }  
  
    then(onFulfilled, onRejected) {  
        if (this.state === 'fulfilled') {  
            onFulfilled(this.value);  
        }  
        if (this.state === 'rejected') {  
            onRejected(this.reason);  
        }  
  
        if (this.state === 'pending') {  
            this.onFulfilledCallbacks.push(() => onFulfilled(this.value));  
            this.onRejectedCallbacks.push(() => onRejected(this.reason));  
        }  
    }  
}  
  
// 使用示例  
const promise = new MyPromise((resolve, reject) => {  
    setTimeout(() => {  
        resolve('成功');  
        // reject('失败');  
    }, 1000);  
});  
  
promise.then(  
    value => console.log(value), // 输出 '成功'  
    reason => console.error(reason)  
);

# 2. await

  • 用于等待一个同步/异步方法执行完成(同步等待)，await必须在async方法中才可以使用。
代码示例
function testA() {
  return 'hello world';
}

async function testA2() {
  return 'xxxx';
}

async function testA3() {
  // await 须在 async函数中才可使用

  // 等待普通函数返回 'heelo world'
  const a1 = await testA();

  // 待会返回promise对象的resolve的值，才会继续往下执行
  const a2 = await testA2();

  console.log(a1 + '--' + a2); // 会打印 hello world--xxxx
}

testA3();

  • 发布/订阅模式
// 定义一个简单的事件总线
class EventBus {
    constructor() {
        this.handlers = {};
    }
 
    // 订阅事件
    subscribe(event, handler) {
        this.handlers[event] = this.handlers[event] || [];
        this.handlers[event].push(handler);
    }
 
    // 发布事件
    publish(event, data) {
        if (!this.handlers[event]) {
            return;
        }
        this.handlers[event].forEach(handler => {
            handler(data);
        });
    }
}
 
// 使用示例
const eventBus = new EventBus();
 
// 订阅事件
eventBus.subscribe('user-logged-in', (user) => {
    console.log(`欢迎 ${user.name} 登录`);
});
 
// 发布事件
eventBus.publish('user-logged-in', { name: 'Alice' });
