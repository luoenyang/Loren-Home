# js 数组

# 增

  • push() 原始数组会改变
  • unshift() 原始数组会改变
  • concat() 数组间的的合并，不影响原始数组

# 删

  • pop() 原始数组会改变
  • shift() 原始数组会改变
  • splice() 原始数组会改变
  • slice() 不影响原始数组

# 改

  • splice()

# 查

  • indexOf() 查找下标
  • includes() 查找值,array.includes(v),返回true/false
  • find() 返回第一个符合的元素
  • findIndex() 返回第一个符合的元素的下标

# 常用方法

  • some() 是否有一个满足，返回true/false
  • every() 是否都满足，返回true/false
  • forEach 处理每一项，不返回新数组，影响原始数组
  • for
  • for value of arrs 用于遍历可迭代对象，value是数组的每一项，不是下标
  • map 处理每一项，返回新数组，不会改变原数组的，但属于浅拷贝，只复制引用
  • filter 返回新数组，不会改变原数组的，但属于浅拷贝，只复制引用

# 排序 原始数组会改变

  • arr.sort(function(a,b){return b-a})
  • arr.reverse() 倒序

# 去重

代码示例
  • reduce可以对数组做迭代操作
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((accumulator, currentValue,index,arr) => accumulator + currentValue, 0);
console.log(sum); // 输出：15

//  检测数组 ages 的所有元素是否都大于 18 ，省去了for循环

    return age >= 18;

ages.findIndex(checkAdult);      // 0

# js 对象的处理

  • for in
  • Object.keys(obj)
const obj={a:1,b:2}
Object.keys(obj); // ['a','b']

  • Object.entries(),返回一个给定对象自身可枚举属性的键值对数组（不包括继承的属性）。
const obj = { a: 1, b: 2, c: 3 };
Object.entries(obj).forEach(([key, value]) => {
    console.log(key, value);
});

const obj = { a: 1, b: 2, c: 3 };
Object.getOwnPropertyNames(obj).forEach(key => {

# 判断数组的方法

  • arr.isArray
  • arr.instanceof Array
  • Object.prototype.toString.call(arr)==="[Object Array]"

# 判断函数的方法

  • typeof fun ==="function"
  • fun.instanceof Function
  • Object.prototype.toString.call(fun)==="[Object Function]"
