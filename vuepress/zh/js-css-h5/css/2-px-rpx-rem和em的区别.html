各单位区别

# 为什么要使用 em 和 rem ？

答：当用户在浏览器中浏览我们制作的 Web 页面时，如果改变了浏览器的缩放，这时会使用我们的 Web 页面布局被打破。

# px rpx em rem pt 区别

区别

  1. px是绝对单位

  2. rpx 在WXSS中，引入了rpx： 1rpx = 375（屏幕宽度-根据手机） / 750（物理像素-固定值） px = 0.5px

  3. em,rem是相对单位 ，Font-size=62.5%，10px = 1em 16px = 1.6em；

  4. em相对于父元素，rem相对于根元素(html)

  5. pt是绝对单位主要用于打印机。

  6. vw：viewpoint width，视窗宽度，1vw等于视窗宽度的1%。

  7. vh：viewpoint height，视窗高度，1vh等于视窗高度的1%。

  8. vmin：vw和vh中较小的那个。ie9 使用vm 代替 vmin。

  9. vmax：vw和vh中较大的那个。

# 使用注意

    1. body选择器中声明Font-size=62.5%或者是10px,所有浏览器默认的都是16px；
    1. 将你的原来的px数值除以10，然后换上em作为单位，10px = 1em 16px = 1.6em；
    1. 重新计算那些被放大的字体的em数值。避免字体大小的重复声明。
html{
	font-size:28px;
}
body{
	font-size: 62.5%;   =  28*62.5%px   如果html不设置则 = 10px
}
.p1{
	font-size: 1em;     =  28*62.5%px
}
.p2{
	font-size: 1rem;    = 28px
}
