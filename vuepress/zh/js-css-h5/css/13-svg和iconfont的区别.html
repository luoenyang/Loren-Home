# svg和字体图标的区别

# 字体图标
  • 字体渲染
  • 单色 （改动颜色只需给字体颜色即可）
  • 矢量图标，就像字体一样可改大小
# svg
  • 图形渲染
  • 多色 （改动颜色、大小相对麻烦）
  • 可访问性好、兼容性好、矢量功能、可读性好 （相对iconfont 锯齿很少）
  • 文本内容可以直接被浏览器，有利于SEO
  • 性能相对有优势，可维护性（当作可使用编辑器修改的图片）
# 总结：
  • 如果不需要经常改动颜色，大小等，就当作图片来使用的，可先考虑svg

第二步：.svg 文件放到svg目录

第三步：编写组件 ../src/components/svgIcon/index.vue

<template>
  <!-- 重点 -->
  <svg :class="svgClass" aria-hidden="true" v-on="$listeners">
    <title>{{title}}</title>
    <desc>{{desc}}</desc>
    <!-- 重点 -->
    <use :xlink:href="iconName" />
  </svg>
</template>

<script>

export default {
  name: 'SvgIcon',
  props: {
    title: {
      type: String,
      required: false
    },
    desc: {
      type: String,
      required: false
    },
    iconClass: {
      type: String,
      required: true
    },
    className: {
      type: String,
      default: ''
    }
  },
  computed: {
    iconName() {
      return `#icon-${this.iconClass}`
    },
    svgClass() {
      if (this.className) {
        return 'svg-icon ' + this.className
      } else {
        return 'svg-icon'
      }
    }
  }
}
</script>
<style scoped>
.svg-icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

const requireAll = requireContext => requireContext.keys().map(requireContext)
requireAll(req)
