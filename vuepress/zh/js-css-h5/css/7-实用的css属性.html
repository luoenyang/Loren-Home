# image的object-fit属性

查看样式

固定容器放置图片很实用 最常用 object-fit: scale-down 、 object-fit: cover

/* 填充- 不保证保持原有的比例，内容拉伸填充整个内容容器 */
object-fit: fill; 
/* 包含- 保持原有尺寸比例。内容被缩放。*/
object-fit: contain; 
/* 覆盖- 保持原有尺寸比例。但部分内容可能被剪切。 */
object-fit: cover; 
/* 不做任何处理 */
object-fit: none; 
/* 缩小- 保持原有尺寸比例。内容的尺寸与 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些。 */
object-fit: scale-down;
