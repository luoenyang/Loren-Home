关键点解释:

  • inheritAttrs: false：禁用默认的属性绑定，这样你就可以手动控制哪些属性传递给内部的 el-input 组件。
  • v-bind="$attrs" 和 v-on="$listeners"：这会将所有传递给 CustomInput 的属性和事件监听器绑定到内部的 el-input 组件上。
  • 使用 v-model 时，Vue 会自动绑定 value 属性和 input 事件。因此，需要在 CustomInput 组件中处理 input 事件并触发相应的更新。

# 1. 封装一个input组件，自动继承el-input的属性和方法

封装CustomInput组件
<template>
  <el-input
    v-bind="$attrs"
    v-on="$listeners"
    :value="value"
    @input="handleInput"
    @change="handleChange"
    <!-- 这里可以添加额外的自定义属性或插槽 -->
  >
    <slot></slot>
  </el-input>
</template>

<script>
export default {
  name: 'CustomInput',
  inheritAttrs: false, // 禁用默认的属性绑定，以便手动控制
  props: {
    value: {
      type: [String, Number],
      default: ''
    },
    // 你可以在这里添加额外的自定义属性
  },
  methods: {
    handleInput(value) {
      this.$emit('input', value);
      // 可以在这里添加额外的处理逻辑
    },
    handleChange(value) {
      this.$emit('change', value);
      // 可以在这里添加额外的处理逻辑
    },
    // 你可以在这里添加额外的自定义方法
  },
  // 你可以在这里添加自定义的计算属性、生命周期钩子等
};
</script>

<style scoped>
/* 你可以在这里添加自定义样式 */
</style>

# 2. 使用组件

使用CustomInput组件
<template>
  <div>
    <custom-input
      v-model="inputValue"
      placeholder="请输入内容"
      @focus="handleFocus"
      @blur="handleBlur"
    ></custom-input>
  </div>
</template>

<script>
import CustomInput from '@/components/CustomInput.vue';

export default {
  components: {
    CustomInput
  },
  data() {
    return {
      inputValue: ''
    };
  },
  methods: {
    handleFocus() {
      console.log('Input focused');
    },
    handleBlur() {
      console.log('Input blurred');
    }
  }
};
</script>

/* 你的样式 */
</style>

# 扩展：封装vue组件时 inheritAttrs 的作用

inheritAttrs属性允许Vue组件在不显式声明props的情况下，使用父组件传递的属性。

inheritAttrs 选项用于控制 Vue 组件是否自动将父组件的属性转换为自身的属性，默认值为 true。当设置为 false 时，父组件的属性不会作为 DOM 属性添加到子组件的根元素上。

默认属性绑定

父组件

<!-- 父组件 -->
<template>
  <div>
    <child-component class="parent-class" :title="parentTitle" />
  </div>
</template>
 
<script>
import ChildComponent from './ChildComponent.vue';
 
export default {
  components: {
    ChildComponent
  },
  data() {
    return {
      parentTitle: 'Parent Title'
    };
  }
};
</script>
