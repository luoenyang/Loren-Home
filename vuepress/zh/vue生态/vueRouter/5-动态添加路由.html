# 动态路由

  • 动态路由是指路由器能够自动的建立自己的路由表，能够根据实际情况的变化实时地进行调整。
  • 动态路由匹配本质上就是通过url进行传参。

比如在写一个商品详情页面的时候，我们的页面结构都一样，只是渲染的数据不同而已，这时候就可以根据商品的不同id去设置动态路由，只需要写一个组件，就可以把每个商品的商品详情映射到同一个组件上去。

    { 
        path: '/Jqxq/:id', // 路由配置拼接
        name: 'Jqxq',
        component: Jqxq
    }
    // 跳转 this.$router.push('/Jqxq/'+ item.id)
    // 接收  ： this.$route.params.id

# 异步路由

异步路由是一种用于延迟加载路由组件的方法。

  • 通过异步路由，只有当路由被访问时，相关的组件才会被加载，从而减少初始加载时间，提高应用的性能和响应速度。
  • 异步路由还可以将应用拆分成更小的模块，便于维护和优化。
  • 在Vue路由配置中，可以为每个路由定义一个异步组件，这个异步组件可以是一个返回Promise的函数，或者是一个使用import()函数动态导入的模块。
查看
// main.js
import Vue from 'vue'
import Router from 'vue-router'

Vue.use(Router)

const routes = [
  {
    path: '/',
    name: 'Home',
    // 使用动态import语法来异步加载组件
    component: () => import(/* webpackChunkName: "home" */ '@/components/Home.vue')
  },
  {
    path: '/about',
    name: 'About',
    // 同样使用动态import语法
    component: () => import(/* webpackChunkName: "about" */ '@/components/About.vue')
  }
  // 你可以在这里添加更多的路由
]

const router = new Router({
  mode: 'history', // 使用HTML5历史模式
  base: process.env.BASE_URL,
  routes
})

export default router
