# keepalive 组件

主要用于保留组件状态或避免重新渲染。

# 原理：

‌缓存机制‌：当使用Keep-alive包裹一个组件时，该组件在切换到后台时不会被销毁，而是被保存在一个缓存对象中（在Vue 2中是JS对象，在Vue 3中是Map对象）。当组件再次被激活时，会直接从缓存对象中恢复实例，而不是重新创建‌。

Keep-alive实际上不是一个真实组件，他没有真实的dom，它是一个抽象组件，用来保存和处理需要缓存的组件的抽象逻辑的：虚拟节点、数据等，节点重新被激活时组件也不会执行mounted生命周期钩子。

# 1. props

  • include - 字符串或正则表达式。只有名称匹配的组件会被缓存。
  • exclude - 字符串或正则表达式。任何名称匹配的组件都不会被缓存。
  • max - 数字。最多可以缓存多少组件实例。

# 2. 用法：保留 a，b 组件状态（缓存）

  • component 组件作用是： 渲染一个“元组件”为动态组件。依 is 的值，来决定哪个组件被渲染
<!-- 逗号分隔字符串 -->
<keep-alive include="a,b">
  <component :is="componentName"></component>
</keep-alive>

<!-- 正则表达式 (使用 `v-bind`) -->
<keep-alive :include="/a|b/">
  <component :is="componentName"></component>
</keep-alive>

<!-- 数组 (使用 `v-bind`) -->
<keep-alive :include="['a', 'b']">
  <component :is="componentName"></component>
</keep-alive>

# 动态组件

使用<component>标签和is属性来实现这一点。

基本用法

使用点击切换和计算属性
<template>
  <div>
    <button @click="currentComponent = 'ComponentA'">Show Component A</button>
    <button @click="currentComponent = 'ComponentB'">Show Component B</button>
    <component :is="currentComponent"></component>
  </div>
</template>

<script>
import ComponentA from './ComponentA.vue';
import ComponentB from './ComponentB.vue';

export default {
  data() {
    return {
      currentComponent: 'ComponentA'
    };
  },
  components: {
    ComponentA,
    ComponentB
  }
};
</script>

计算属性

<template>
  <component :is="computedComponent"></component>
</template>

<script>
import ComponentA from './ComponentA.vue';
import ComponentB from './ComponentB.vue';

export default {
  data() {
    return {
      someCondition: true
    };
  },
  computed: {
    computedComponent() {
      return this.someCondition ? 'ComponentA' : 'ComponentB';
    }
  },
  components: {
    ComponentA,
    ComponentB
  }
};
</script>

# 异步组件，动态加载组件

import 动态加载组件，而不是在初始加载时就导入所有组件。

异步组件
<template>
  <component :is="asyncComponent"></component>
</template>

<script>
export default {
  data() {
    return {
      asyncComponent: () => import('./ComponentA.vue')
    };
  }
};
</script>

# 继承组件

在Vue中，可以通过局部注册和全局注册的方式来继承一个组件。

局部注册：
在父组件中通过 import 导入子组件，然后在父组件的 components 选项中注册。

// 子组件 Child.vue
<template>
  <div>我是子组件</div>
</template>
 
// 父组件 Parent.vue
<template>
  <div>
    <child></child>
  </div>
</template>
 
<script>
import Child from './Child.vue';
 
export default {
  components: {
    Child
  }
}
</script>

全局注册： 如果你想在多个组件中使用同一个组件，可以在Vue实例之外全局注册，这样就不必在每个组件中都导入和注册了。

import Vue from 'vue';
import Child from './Child.vue';
 
Vue.component('child', Child);
 
// 之后在任何组件中都可以直接使用 <child></child>

# 组件的设计原则

要考虑以下方面：

  • 复用性
  • 扩展性
  • 易用性
  • 接口清晰
  • 输入props
  • 输出，emits及页面的呈现
  • 要低耦合，组件间的依赖尽可能的少
