# nvm 是什么

nvm-windows： nodejs多版本的控制，可安装和随意切换已安装的nodejs版本来使用

官网地址 https://github.com/coreybutler/nvm-windows/releasesopen in new window

# 下载安装

下载下图列表的安装文件来安装可免去手动配置环境

# 配置淘宝镜像

配置settings.txt：在安装nvm的目录找到 settings.txt 文件，添加以下代码

node_mirror: https://npm.taobao.org/mirrors/node/

npm可以不设置淘宝镜像，否则使用npm可能会出现版本不符合的提示
npm_mirror: https://npm.taobao.org/mirrors/npm/

# 常用nvm命令


# 查看已安装的nodejs列表，和当前使用的nodejs（前面带*）
nvm ls

# 安装nodejs
nvm install v16
nvm install v16.10.22

# 切换使用nodejs
nvm use v16.10.22
nvm use 16.10.22

# 显示当前的版本
nvm current 

# 删除已安装的指定版本，语法与install类似
nvm uninstall v16.10.22
