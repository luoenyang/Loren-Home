# Buffer类

  • JavaScript 语言自身只有字符串数据类型，没有二进制数据类型。
  • 当操作文件流时需使用二进制，node定义Buffer类就是专门存放二进制数据的缓存区。
代码示例
// 创建buffer
var b= new Buffer("someString", "utf-8");

// 写入buffer
b.write("otherString",0,10,"utf8");

// 读取buffer数据
b.toString('utf8',0,5)

// 转化为json
b.toJSON();

// 合并
b1.concat(b2);

// 拷贝
b.copy();

// 裁剪
b.slice();

// 比较
b1.compare(b2)

# Stream流

Stream是一个抽象接口，http请求request 对象就是一个 Stream。 一般用于文件的传输。
4种类型：可读，可写，可读可写，转换

事件：

  1. data - 当有数据可读时触发。
  2. end - 没有更多的数据可读时触发。
  3. error - 在接收和写入过程中发生错误时触发。
  4. finish - 所有数据已被写入到底层系统时触发。
读取流数据
// 引入模块
var fs = require("fs");

// 存放数据
var data = "";

//创建可读流
var readerStream = fs.createReadStream("/test.txt");

//设置编码
readerStream.setEncoding('UTF8');

//处理事件流
readerStream.on('data',function(s){
    data+=s;
});
readerStream.on('end',function(){});
readerStream.on('error',function(error));
readerStream.on('finish',function());

# 管道流

通常我们用于从一个流中获取数据并将数据传递到另外一个流中。

代码示列
// 引入模块
var fs = require("fs");

//创建一个可读流
var readerStream = fs.createReadStream("../get/test.txt");

//创建一个可写的流
var writerStram = fs.createWriteStream("../set/test.txt");

// 管道的读写操作，把读取的写到可写的流中
readerStream.pipe(writerStram);

# 链式流

链式是通过连接输出流到另外一个流并创建多个对个流操作链的机制。

代码示列
// 引入模块
var fs = require("fs");
var zlib = require('zlib');

// 压缩 
// test.txt 压缩文件为 test.txt.gz
var r = fs.createReadStream('/test')
    .pipe(zlib.createGzip())
    .pipe(fs.createWriteStream('/test.txt.gz'));



// 解压
// 压缩完立即解压
r.on('finish',function(){
   console.log("文件压缩完成！");
   console.log("现在开始解压！");

   // test.txt.gz 解压文件为 test.txt
   var d = fs.createReadStream('/test.txt.gz')
       .pipe(zlib.createGunzip())
       .pipe(fs.createWriteStream('/test.txt'));
   d.on('finish',function(){
      console.log("文件解压完成。");
   })
});
