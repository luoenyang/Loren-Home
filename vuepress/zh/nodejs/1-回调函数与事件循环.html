# 回调函数

是Nodejs异步编程的体现。Nodejs 使用了大量的回调函数。
异步I/O，一边读取文件，一边执行其他命令，在文件读取完成后，将文件内容作为回调函数的参数返回。

代码示例
//载入fs模块 并把实例化后的fs赋值给变量fs
var fs = require("fs");
fs.readFile('../txt/callback', function (err, data) {
    // ...
});

// 其它操作
// ...

# 模型、循环

nodejs 是单进程单线程应用程序，但是通过事件和回调函数支持并发。
事件驱动模型中，会有一个主循环来监听事件，当检测到事件时触发回调函数。

# 事件绑定、触发

绑定：eventEmitter.on('eventName',callback)
触发：eventEmitter.emit('eventName')

代码示例

//引入事件模块 events
var events = require('events');

//创建 eventEmitter 对象
var eventEmitter = new events.EventEmitter();

//创建一个事件处理程序
var connectHandler = function connected() {
    console.log('连接成功！');

    // 在此执行数据接收事件， 这是一个nodejs的内置事件
    eventEmitter.emit('data_received');
};


// 绑定 connection 事件，这也是一个nodejs内置事件
// connectHandler 为绑定后的执行程序
eventEmitter.on("connection", connectHandler);

// 绑定 data_received 事件，这也是一个nodejs内置事件
eventEmitter.on("data_received", function () {
    console.log("接收数据成功！")
});
