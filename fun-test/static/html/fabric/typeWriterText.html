Fabric.js 打字机效果动画

中速

这个示例展示了如何使用Fabric.js创建打字机效果的动画。文本会逐字显示，并带有光标闪烁效果。

您可以通过上方的控件调整打字速度，并输入自定义文本。

核心代码逻辑：

// 创建文本对象
const text = new fabric.Text('', {
    left: 50,
    top: 100,
    fontSize: 24,
    fill: 'white',
    fontFamily: 'Courier New'
});

// 打字机动画函数
function typeWriter() {
    if (currentIndex < fullText.length) {
        text.set('text', fullText.substring(0, currentIndex + 1));
        canvas.renderAll();
        currentIndex++;
        typeTimeout = setTimeout(typeWriter, typingSpeed);
    } else {
        isTyping = false;
        startBtn.disabled = false;
        pauseBtn.disabled = true;
    }
}
