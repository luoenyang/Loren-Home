一. 音频速度变化、波形图 的解决方案：使用wavesurfer.js,可以选择视频也可以选择音频

二. 缺点：使用backend:'WebAudio'的方式，音量可以大于1，但变速后音调会变（webav采取的方式，音量小于1速度不等于1也会变调）

三. 缺点：使用backend:'MediaElement'的方式，音量最大是1（原声），变速后音调不变（audio的标签是这样的机制，不变调，但音量不能大于1）

速度、音量大于1不变调 的解决方案：

1. 采用ffmpeg进行预处理，把音量调高生成一个临时文件再处理： ffmpeg -i 说话声音.mp3 -af "volume=10" bigVolume.mp3

2. 使用video标签且音频分离，this.audioContext = new (window.AudioContext || window.webkitAudioContext)();的方式播放

3. 使用 import { SoundTouchNode } from "soundtouchjs"; 进行播放 效果类似2;详见mediabunny.vue
